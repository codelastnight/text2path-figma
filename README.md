# To-Path: A Figma Plugin


Put any objects or text on a path! 

Select an object and a curve, adjust the options, and watch the magic* happen!

release 1.0 



## development setup

1.  install packages:
`npm i` 

2. compile :
`npm run prod`

Helpful resources:
